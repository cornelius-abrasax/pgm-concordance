# pgm-concordance
A reference concordance of deities, divine beings, and sacred names invoked in the Papyri Graecae Magicae, drawn from the Preisendanz corpus and Betz's translation. Names are cross-cultural, reflecting the syncretic Greco-Egyptian milieu of Roman Egypt. Also includes ability to browse the complete PGM and PDM corpus by magical theme.

Note that the HTML was created with Claude so likely could be better. It's mostly here to use the Pages hosting feature so magicians and interested others can use the mini-site.
